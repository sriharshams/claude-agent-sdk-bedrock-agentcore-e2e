# Customer Support Domain Knowledge

## Product Categories
- **Smartphones**: 30-day return window, 1-year warranty, factory reset required
- **Laptops**: 30-day return window, 1-year warranty + extended options, tech support verification needed
- **Accessories**: 30-day return window, 90-day warranty, unopened preferred
- **Headphones**: 1-year warranty, firmware updates via companion app
- **Monitors**: 3-year warranty, color calibration support

## Return Policy Guidelines
- All categories have a 30-day return window from delivery
- Original packaging and all accessories must be included
- Refund timelines vary: 3-10 business days after inspection
- Free return shipping for most categories over $50

## Warranty Procedures
1. Verify warranty status using `check_warranty_status` tool with serial number
2. Confirm customer email matches warranty records
3. Calculate days remaining and recommend appropriate action
4. For expired warranties, suggest extended coverage options

## Escalation Rules
- If customer reports a safety issue, immediately escalate
- If customer has been transferred more than twice, escalate to supervisor
- If the issue cannot be resolved with available tools, provide direct support contact
- Complex warranty disputes should be handled by the warranty department

## Tone Guidelines
- Be friendly, patient, and empathetic
- Use the customer's name when available
- Acknowledge frustration before providing solutions
- Always offer additional help after answering questions
- Avoid technical jargon unless the customer uses it first

## Common Scenarios
1. **Return request**: Check return policy, verify eligibility, guide through RMA process
2. **Warranty check**: Use warranty tool, verify serial number, provide status and recommendations
3. **Product inquiry**: Use product info tool, provide specs and compatibility details
4. **Technical issue**: Search knowledge base, provide troubleshooting steps
5. **General question**: Use web search for current information
